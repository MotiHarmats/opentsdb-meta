Builds the MystSegment. Order of the segment is as follows.

  1. Metric Bitmaps
  2. Tag Keys Bitmaps
  3. Tag Values Bitmaps
  4. Epoch Bitmaps
  5. Epoch Bitmap header
  6. Dictionary
  7. Docstore
  8. Docstore header
  9. FST
  10. FST Header
  11. Myst Segment Header The offset before start of each structure is written to header MystSegmentHeader
